Division Event Logger Privacy Policy

This Privacy Policy outlines how the Division Event Logger bot collects, uses, and shares information from users and Discord servers. By using the bot, you agree to the data practices described below.

1. Data We Collect

The Division Event Logger may collect the following types of data:

  • User Attendance: Information about which users attend events and the times they attend.
  • Server Presence: Data about which servers a user and the bot are both in.
  • User Invitations: Information on who invited a user to a specific server.
  • Server Settings: Configurations and settings that a server has customized for the botâ€™s functionality.
  • Event Statistics: Data on user and server activities, including the number of events attended, hosted, and average attendance, as well as total event attendees, recruitment efforts, and similar statistics.
  • Roblox Group Association: Information about which Roblox group is linked to a specific Discord server, including details for rank tracking and promotions.
  • Channel-Function Links: Information about which Discord channels are linked to specific bot functions, such as where event logs should be sent or other functional integrations.
  • Role-Rank Links: Information about which Discord roles are associated with specific Roblox ranks for automated rank management and promotion.

2. How We Use the Data

The data collected by Division Event Logger is used for the following purposes:

  • Event Tracking: To log user and server event participation for accurate event reporting.
  • Bot Functionality: To provide personalized and effective services based on server settings and user interactions.
  • Data Analysis and Aggregation: To derive additional statistics based on collected data. This may include calculations such as total event attendance, average event participation, user recruitment effectiveness, server-wide statistics (e.g., total events hosted, total attendees), and other metrics. These derived metrics may be used to generate leaderboards and other comparative displays.
  • Leaderboards and Public Statistics: To display rankings and statistics for both users and servers, such as total events hosted, average attendance, user recruitment, and experience points (XP). These statistics may be shared within the Discord server through public leaderboards or other displays.
  • Roblox Group Association and Promotions: To track Roblox group membership and promote attendees who have reached a new rank based on event participation or other criteria. This includes the automated promotion of users within Roblox groups based on rank progression.
  • Role-Rank Links: To manage and update user roles in Discord based on their Roblox rank, ensuring consistency between Roblox ranks and corresponding Discord roles.
  • Channel-Function Links: To ensure that specific bot functions (such as sending event logs or other notifications) are directed to the correct Discord channels as configured by the server owner.

3. Data Sharing

We do not share, sell, or trade any personal or server data with third parties, except in the following circumstances:

  • With Your Consent: We will only share data if you give explicit consent.
  • Legal Compliance: If required by law, we may disclose data in response to a valid legal process.
  • Public Leaderboards and Statistics: User and server statistics such as event attendance, hosting, recruitment, average attendance, and experience points (EXP) may be displayed publicly within the Discord server in the form of leaderboards or other statistical summaries.

4. Data Retention

Data is stored for as long as necessary to provide the services offered by the bot. Server owners may request the deletion of specific data by contacting the bot owner.

5. Changes to This Policy

We may update this Privacy Policy from time to time. When changes are made, we will notify users within the bot or through a server announcement.

6. Contact Us

If you have any questions or concerns about this Privacy Policy or the data collected by the Division Event Logger, please contact the bot owner Ererej on discord.
